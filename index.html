Membuat Aplikasi Mobile Android Tema Perpustakaan Menggunakan Android Studio

Sebuah Project Ulangan Akhir Semester 5 Mata Kuliah Pemrograman Mobile yang ditugaskan untuk membuat sebuah aplikasi mobile android menggunakan Android Studio. Pada proyek UAS ini, setiap mahasiswa diberikan tugas dan tema masing-masing untuk membuat fitur pada bagian halaman aplikasi Perpustakaan. Saya mendapatkan tema Login, yang mengharuskan saya untuk membuat sistem CRUD untuk menampilkan, menambahkan, mengedit, dan menghapus data login. Sistem ini dirancang untuk memudahkan pengelolaan informasi login di perpustakaan, sehingga admin dapat dengan mudah mengatur data login yang sudah ada. Aplikasi ini diharapkan dapat membantu operasional perpustakaan dalam mengelola data login dengan lebih efisien dan terorganisir.

Membuat Web CRUD Multi Account User Sederhana Menggunakan PHP Native dan Bootstrap 5

Sebuah Project Ulangan Akhir Semester 4 Mata Kuliah Pemrograman Web yang ditugaskan untuk membuat sebuah Web sistem informasi Multi Account User yang mengubah tampilan sesuai dengan level akun user yang didaftarkan. Pada proyek UAS ini, Saya memilih tema web berupa jual beli barang mahasiswa yang mana ada tampilan operator barang, tampilan mahasiswa dan tampilan admin. akun user operator barang tidak akan menampilkan tampilan halaman mahasiswa begitu juga sebaliknya namun untuk akun user admin bisa dapat melihat kedua tampilan akun user tersebut dan juga bisa membuat / registrasi akun baru dan memberikan level akun usernya. dalam pembuatan web ini saya menggunakan bahasa pemrograman php native dan juga untuk tampilan menggunakan template Bootstrap 5

CRUD HTTP REST-API untuk Web Multi Account User menggunakan Aplikasi POSTMAN

Masih dalam Project Ulangan Akhir Semester 4 Mata Kuliah Pemrograman Web, saya membuat sebuah Web Sistem Informasi Multi Account User yang dicoba CRUD melalui HTTP REST API menggunakan aplikasi POSTMAN. Dalam proyek ini, saya mengimplementasikan berbagai fitur untuk mengelola akun pengguna, termasuk pembuatan, pembaruan, penghapusan, dan penampilan data pengguna. Pengujian dilakukan menggunakan POSTMAN untuk memastikan bahwa semua endpoint API berfungsi dengan benar dan dapat menangani permintaan HTTP dengan baik. Tujuan dari project ini adalah untuk mengembangkan keterampilan dalam membuat dan menguji API, serta mengelola data pengguna secara efisien dalam sebuah sistem informasi berbasis web.

Project ini memprediksi kapan bermain golf. dalam pelatihan ini diharapkan dapat meningkatkan kemampuan teknis dalam mengalisis data/mengolala.
